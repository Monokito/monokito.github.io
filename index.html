Why is Monokito.sync keeping this website so basic?

The reason why we keep the site so basic, is that we like the web 1.0 esthetic so much. Besides that, a site is more clear when it is text based and has the relevant links on it. It is nice to see that one can host a website for free, that way it is not necessary to pay a zillion dollards a month. We can deliver our message and share video, music and text with you guys.
